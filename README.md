# CodeForVSC

这个仓库，emm，是最开始用vscode编写代码时创建的，内容应该主要是算法代码，也就是大学阶段准备acm和刷leetcode求职的代码，也包含一些乱七八糟的内容。

因为所有文件都在根目录，多种语言的文件加载在一起，管理太过混乱了，所以打算将其封存起来算了，之后打算为不同的语言/学习方向单独创建vscode工作区，开新的仓库去维护
